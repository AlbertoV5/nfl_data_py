# nfl_data_py

nfl_data_py is a Python library for interacting with NFL data.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install nfl_data_py.

```bash
pip install foobar
```

## Usage

```python
import nfl_data_py
```
